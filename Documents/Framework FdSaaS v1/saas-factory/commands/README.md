# Slash Commands (15x)

Quick-access commands for SaaS Factory orchestration.

## Commands List

1. `/start-sprint` - Inicia nuevo sprint para el producto activo → `start-sprint.md`
2. `/switch-product` - Cambia producto activo (archiva actual, carga target) → `switch-product.md`
3. `/validate-scope` - QA Auditor verifica scope creep del MVP → `validate-scope.md`
4. `/archive-session` - Cierra sprint y archiva estado del producto → `archive-session.md`
5. `/health-check` - Audita estado técnico del producto activo → `health-check.md`
6. `/idea-refiner` - Refina ideas crudas con filtros Hormozi/Naval/Jobs → `idea-refiner.md`
7. `/assign-agent` - Rutea tarea a skill específico
8. `/review-code` - Code review + validación de tests
9. `/check-compliance` - Auditoría PCI/WCAG/OWASP
10. `/market-research` - Activa Investigador de Mercado
11. `/design-review` - Validador Interfaz — componentes y UX
12. `/deploy-staging` - Deploy a staging environment
13. `/generate-report` - Reporte de estado del producto
14. `/escalate` - Escalar blocker crítico
15. `/sync-decisions` - Sincronizar decisions.md
16. `/help` - Referencia de todos los comandos

## Usage Pattern

Each command updates `context/session-log.md` with timestamp + outcome.
