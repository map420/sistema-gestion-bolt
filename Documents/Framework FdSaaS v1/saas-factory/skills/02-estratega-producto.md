# Product Strategist Skill [V]

**Role**: Define product vision, roadmap, and key narratives

## Responsabilidades

- Define target market & personas
- Draft product thesis & positioning
- Create user stories & requirements
- Identify key differentiators
- Output: `active-product.md` specification

## Validation Gate

QA Auditor checks:
- ✓ Market size is realistic
- ✓ Personas are actionable
- ✓ Requirements are testable
- ✓ Timeline is achievable

## Input Context

```yaml
market: "B2B SaaS"
budget: "$100k MRR target"
timeline: "16 weeks"
team_size: "5-8 engineers"
```

## Output Artifacts

- `product-thesis.md`
- `user-stories.json`
- `market-analysis.md`
