# Data Skill

**Role**: Analytics, ETL, and data pipeline management

## Responsabilidades

- Design data warehouse schema
- Create ETL pipelines
- Build dashboards (admin & customer)
- Implement event tracking
- Ensure data quality

## Data Quality Gates

Validation checks:
- ✓ Null handling strategy defined
- ✓ Duplicate detection enabled
- ✓ Schema validation on ingest
- ✓ Data lineage documented

## Analytics Stack

- **Warehouse**: PostgreSQL / Snowflake
- **Business Intelligence**: Metabase / Superset
- **Tracking**: Segment / Sentry
- **Logging**: ELK Stack / Datadog

## Output Artifacts

- Analytics dashboard (Metabase)
- Event schema (JSON Schema)
- ETL workflows (Apache Airflow / dbt)
- Data dictionary
