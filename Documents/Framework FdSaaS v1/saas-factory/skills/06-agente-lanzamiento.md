# Launch Skill [V]

**Role**: Go-to-market strategy, marketing, and deployment coordination

## Responsabilidades

- Create GTM strategy & timeline
- Plan PR & content calendar
- Set up analytics dashboards
- Coordinate production deployment
- Monitor launch metrics

## Pre-Launch Checklist

Validation gates:
- ✓ Security audit passed (Snyk, OWASP)
- ✓ Load testing successful (100K req/min)
- ✓ Rollback plan documented
- ✓ Customer support scripts ready
- ✓ Monitoring & alerts configured

## Launch Timeline

- **T-4w**: GTM plan finalized
- **T-2w**: Beta access to early users
- **T-1w**: Deployment rehearsal
- **T-0**: Production deployment
- **T+1w**: First retrospective

## Output Artifacts

- GTM plan & timeline
- Press release template
- Customer onboarding flow
- Post-launch metrics dashboard
