# Prompts [V] NUEVO

Brief templates and session starters for agent workflows.

## Prompt Categories

### Session Starters
- `product-kickoff.md` - Begin new product development
- `sprint-planning.md` - Plan weekly sprint
- `retrospective.md` - Post-sprint reflection

### Research Templates
- `market-sizing.md` - Framework for TAM/SAM/SOM analysis
- `competitor-analysis.md` - Template for competitive matrix
- `trend-spotting.md` - Research prompt for industry trends

### Design Prompts
- `component-spec.md` - From feature to React component
- `user-flow.md` - Convert user story to interaction diagram
- `design-system.md` - Establish design tokens

### Technical Prompts
- `api-design.md` - RESTful API specification template
- `database-schema.md` - ER diagram to Prisma schema
- `deployment.md` - Infrastructure as code checklist

### Compliance Prompts
- `pci-checklist.md` - PCI-DSS compliance verification
- `gdpr-checklist.md` - GDPR requirements mapping
- `security-audit.md` - OWASP top 10 checklist

---

## Usage

Each prompt is designed to be copy-pasted into Claude/ChatGPT and executed by the corresponding agent.

Example: `Product Strategist opens market-sizing.md → completes template → outputs to active-product.md`
