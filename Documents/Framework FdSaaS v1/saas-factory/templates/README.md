# Templates [V] NUEVO

Scaffolds and boilerplate for rapid product building.

## Available Templates

### B2B SaaS
- `b2b-saas-starter.zip` - Complete project structure
  - User authentication (OAuth + JWT)
  - Dashboard with organization hierarchy
  - Team collaboration features
  - Subscription management
  - API documentation

### AI-Powered Features
- `ai-chat-service.zip` - Claude integration
  - Message streaming
  - Context management
  - Token counting
  - Error handling

### Data Analytics
- `analytics-dashboard.zip` - Metabase + PostgreSQL
  - Event tracking schema
  - Custom dashboards
  - User cohort analysis

### Payment Integration
- `stripe-subscription.zip` - Stripe + database schema
  - Subscription create/update/cancel
  - Metered billing
  - Webhook handlers

## Quick Start

1. Choose template based on product type
2. `unzip template.zip -d my-product`
3. Follow `SETUP.md` included in template
4. Customize for your use case
5. Deploy to staging
